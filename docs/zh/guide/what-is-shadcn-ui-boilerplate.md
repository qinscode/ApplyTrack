# shadcn/ui boilerplate 是什么

shadcn/ui boilerplate 是一个基于 [**shadcn/ui**](https://github.com/shadcn-ui/ui) 的模板，用于快速开发现代 web 应用程序。


## 使用场景

- 仪表盘
- 管理后台
- 单页应用
- 等等

## 为什么选择 shadcn/ui boilerplate

用不用由你，这只是我个人的一个项目，我需要一个这样的模板来快速开发和部署我的应用程序。
