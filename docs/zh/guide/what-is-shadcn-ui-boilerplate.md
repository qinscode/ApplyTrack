shadcn/ui boilerplate 是一个基于 [**shadcn/ui**](https://github.com/shadcn-ui/ui) 的现代化、可扩展的样板项目，用于快速开发使用 React、Tailwind CSS 和 Radix UI 构建的美丽、响应式的 Web 应用程序。


## 使用场景

- 仪表盘
- 管理后台
- 单页应用
- 等等

## 为什么选择 shadcn/ui boilerplate
